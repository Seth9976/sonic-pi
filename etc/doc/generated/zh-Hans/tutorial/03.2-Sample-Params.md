3.2 示例参数

# 示例参数：放大器和平移

正如我们在合成器上看到的那样，我们可以轻松地通过参数控制声音。示例支持完全相同的参数化机制。让我们重温我们的朋友“ amp：”和“ pan：”。

## 激爽示例

您可以使用与合成器完全相同的方法来更改样本的幅度：

```
sample :ambi_lunar_land, amp: 0.5
```

## 平移样本

我们还可以在样本上使用` pan:`参数。例如，以下是我们先给左耳演奏the amen break，然后再演奏进行到一半时通过右耳再次演奏的方式：

```
sample :loop_amen, pan: -1
sleep 0.877
sample :loop_amen, pan: 1
```

注意0.877（以秒为单位）是`：loop_amen`采样持续时间的一半。

请注意，最后如果您使用`use_synth_defaults`设置了一些合成器默认值（我们将在后面讨论），则它们将被`sample`忽略。
