# 你好，朋友 :-)

歡迎使用 Sonic Pi。我將教你如何創作天籟之音，希望你會喜歡。接下來的學習過程將會*非常有趣*，你將會學習音樂、合成器、程式設計、編曲、演奏等所有相關知識。

哦抱歉，請允許我先自我介紹，我是 [Sam Aaron](http://twitter.com/samaaron)，就是創造出 Sonic Pi 的傢伙。你可以在 Twitter 上找到我 [@samaaron](http://twitter.com/samaaron)，我會很高興認識你的。你可能也對我在觀眾面前用 Sonic Pi 進行的 [Live Coding 表演](https://youtube.com/samaaron) 感興趣。

如果你有任何想法、意見，想要改進 Sonic Pi，請務必告訴我，你的回饋說不定會成為下一個版本的重量級功能！

這份教學分成好幾個章節。我在編寫時盡量由淺入深，讓你可以無痛地從入門到精通，你可以自由從你感興趣的章節開始學習。如果你覺得缺少了某些內容，請告訴我，我會考慮在未來的版本添加進來。

最後，看其他人的 Live coding 表演是一個很棒的學習方式。我通常會在 [http://youtube.com/samaaron](http://youtube.com/samaaron) 現場直播，歡迎收看、跟我打招呼、然後問我一堆問題 :-)

OK，讓我們開始吧…
