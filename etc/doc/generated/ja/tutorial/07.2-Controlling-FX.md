# エフェクトの制御

エフェクトも、少々異なる方法をとりますが、制御する事が出来ます。

```
with_fx :reverb do |r|
  play 50
  sleep 0.5
  control r, mix: 0.7
  play 55
  sleep 1
  control r, mix: 0.9
  sleep 1
  play 62
end
```

変数を使用する代わりに、*do/end*プロックのブロックパラメータを使用します。これは、実行中のエフェクトとして*do/end*プロックの中から参照するパラメータとして、`|`と`|`の間にユニークな名前を指定する必要があります。このパラメータの振る舞いは、引数の付いた関数を使用する場合と同じです。

さあ、シンセやエフェクトをコントロールしてみよう！
